Programming is the art of telling another human what one wants the computer to do. Computer science is no more about computers than astronomy is about telescopes. The best thing about a boolean is even if you are wrong, you are only off by a bit.

There are only two hard things in Computer Science: cache invalidation and naming things. Any fool can write code that a computer can understand. Good programmers write code that humans can understand. First, solve the problem. Then, write the code.
